# Fullstack Authentication App
A fullstack authentication application built using React, TypeScript, NestJS, MongoDB, and JWT authentication.

The app includes user signup, login, protected routes, authentication persistence, and logout functionality.
## Tech Stack

### Frontend
- React
- TypeScript
- React Router
- Tailwind CSS
- Axios

### Backend
- NestJS
- MongoDB
- Mongoose
- JWT Authentication
- Passport.js

## Live Demo

## Installation

### Frontend

```bash
npm install
npm run dev
```

### Backend

```bash
npm install
npm run start:dev
```

# Environment Variables 😎🔥

### Frontend

```env
VITE_API_URL=your_backend_url
```

### Backend

```env
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
PORT=5000
```
